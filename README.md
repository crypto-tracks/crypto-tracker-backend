# Getting Started crypto-tracker-backend
The Backend for the Crypto Tracker

## Running Locally

### `npm start`

Runs the server on [http://localhost:3001](http://localhost:3001)

## Testing Locally

1. Run the server with `npm start`

2. Run Cypress with `npm run test:open`
